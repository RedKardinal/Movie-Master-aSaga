# Movie Master - A Saga!

A basic "NETFLIX" like display of movies for users to search and find information. Users can
click and view written description of each and edit.

## Built With ...

React, JavaScript, Node, Express, Postgresl, and Material-UI

## To Spin Up

1. npm install

2. npm run client

3. npm run server
